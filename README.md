# eBay Scout Website

This repository contains the public static website for eBay Scout.

The scanner/runtime code is private and is not included in this public repository.

## GitHub Pages

The site lives in `website/` and is deployed with GitHub Actions from `.github/workflows/pages.yml`.

Published site:

https://rain0x06.github.io/EbayScout/

## Local Preview

Open `website/index.html` directly in a browser, or serve the folder with any static file server.

## Deployment

GitHub Pages deploys the `website/` folder on every push to `main`.
