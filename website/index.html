Discord setup

Invite the bot. Keep secrets server-side.

Use your Discord application's public Application ID to generate an OAuth invite for your server.

Open Discord Developer Portal
Open official Discord portal

In Discord Developer Portal, create or open your application, then copy General Information > Application ID. That numeric Application ID is the client ID used for the invite link.

The invite uses Discord's Send Messages permission value 2048 automatically.
