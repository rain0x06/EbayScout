Privacy by shape

Useful listing context, without account baggage.

The public site is static and does not collect private runtime credentials. Alerts are shaped around listing context instead of user account records.

No eBay sign-in

End users do not sign in to an eBay account through this website.

No seller profiles in previews

Public examples avoid seller usernames, profile links, and item IDs.

No public secrets

The website does not collect bot tokens, webhooks, client secrets, or API keys.

No buyer/order records

The app does not require addresses, payment data, messages, or transaction records.
